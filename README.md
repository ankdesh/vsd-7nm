# VSD 7nm FinFET Project

This repository contains materials, assignments, and simulations related to 7nm FinFET course.

## Project Structure

### Assignment folder
Contains assignment work and simulation setups.
- **[assign1](assign/assign1)**: Assignment 1
- **[assign2](assign/assign2)**: Assignment 2
- **[SETUP_BSIM-CMG-README.md](assign/SETUP_BSIM-CMG-README.md)**: Instructions for setting up BSIM-CMG for use with Ngspice.
- **[SETUP_NGSPICE.md](assign/SETUP_NGSPICE.md)**: Instructions for NGSPICE setup.

### [Ref_from_vsd](Ref_from_vsd)
Reference materials and initial designs provided by VSD.
- Contains Xschem files and initial spice models.

### [theory](theory)
Theoretical background and course materials from Week 1 lectures.
- **[Scaling Beyond CMOS](theory/Scaling%20Beyond%20CMOS:%20FinFET%20Devices%20and%20Innovations)**: Materials on FinFET devices and innovations.
